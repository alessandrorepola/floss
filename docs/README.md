### Essential Commands
```bash
# Installation

# Quick start
pyfault run --source-dir src
pyfault ui

# Help
pyfault --help
pyfault COMMAND --help
```

- `pyfault.conf` - Configuration file
- `coverage.json` - Test coverage data
- `report.json` - Fault localization results
